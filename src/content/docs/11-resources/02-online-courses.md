## DeepLearning.AI 课程

### Agentic AI with Andrew Ng

**链接：** https://www.deeplearning.ai/courses/

**简介：** Andrew Ng 亲自讲解 Agentic AI 的核心概念和设计模式，包括 Reflection、Tool Use、Planning 和 Multi-Agent 协作四种关键 Agent 模式。

**推荐理由：** Andrew Ng 的讲解一如既往地清晰易懂。这门课是理解 Agentic AI 设计模式的最佳入门资源之一，短小精悍。

---

### Agent Skills with Anthropic

**链接：** https://www.deeplearning.ai/short-courses/

**简介：** 与 Anthropic 合作的实战课程，教你如何使用 Claude 构建具备工具使用能力的 Agent，涵盖 Tool Use、系统提示设计、Agent 循环构建等。

**推荐理由：** 直接使用 Anthropic 的 API 和最佳实践，适合准备在生产中使用 Claude 的开发者。

---

### Building Agentic RAG with LlamaIndex

**链接：** https://www.deeplearning.ai/short-courses/building-agentic-rag-with-llamaindex/

**简介：** 教你使用 LlamaIndex 构建 Agentic RAG 系统，将 RAG 从简单的检索-生成升级为能够自主推理和多步检索的 Agent。

**推荐理由：** 将 RAG 和 Agent 结合是当前最热门的应用方向。LlamaIndex 在这个领域有很强的工具支持。

---

### Multi-Agent Systems with CrewAI

**链接：** https://www.deeplearning.ai/short-courses/multi-ai-agent-systems-with-crewai/

**简介：** 使用 CrewAI 框架构建多 Agent 协作系统，学习角色定义、任务分配、Agent 间通信和结果整合。

**推荐理由：** Multi-Agent 是 Agent 架构的重要方向，CrewAI 提供了直观的框架来理解多 Agent 协作的核心概念。

---

### Functions, Tools and Agents with LangChain

**链接：** https://www.deeplearning.ai/short-courses/functions-tools-agents-langchain/

**简介：** 使用 LangChain 实现 Function Calling 和 Tool Use，构建能够调用外部工具的 Agent。

**推荐理由：** LangChain 是最流行的 Agent 框架之一，这门课帮你快速掌握其核心功能。适合需要快速上手框架的开发者。

---

## Coursera 课程

### Hands-on Agentic AI Specialization

**链接：** https://www.coursera.org/

**简介：** Coursera 上的 Agentic AI 专项课程，从概念到实战，涵盖 Agent 设计、Tool Use、RAG 和部署。包含动手实验和项目。

**推荐理由：** 比短课程更深入，有完整的项目实践。适合想要系统性学习并获得证书的学习者。

## 学习建议

### 推荐学习顺序

```
1. Andrew Ng 的 Agentic AI（概念入门，1-2 小时）
      ↓
2. Functions, Tools and Agents with LangChain（工具使用基础）
      ↓
3. Agent Skills with Anthropic（Anthropic 生态实战）
      ↓
4. Building Agentic RAG with LlamaIndex（RAG + Agent 结合）
      ↓
5. Multi-Agent with CrewAI（进阶架构）
```
